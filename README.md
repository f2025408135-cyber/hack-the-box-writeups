# Hack The Box Writeups

Hey there! I'm a senior penetration tester and technical writer with 8+ years of experience in offensive security. This repository serves as both a public portfolio of my work and a learning resource for the cybersecurity community. I believe in documenting not just the successful commands, but the thought process, the rabbit holes, and the failures that lead to the final shell.

*Disclaimer: These are AI-assisted writeups based on my completion of the machines, intended to reflect real-world pentesting methodologies.*

## Techniques Legend

- **Web Exploitation**: SQLi, XSS, SSRF, LFI, RFI, Command Injection, Deserialization, API logic flaws
- **Active Directory**: Kerberoasting, AS-REP Roasting, DCSync, BloodHound, SMB Relaying, Golden Tickets
- **Binary Exploitation**: Buffer Overflows, ROP chains, Format Strings
- **Container Escapes**: Docker socket abuse, LXD/LXC group privilege escalation
- **Misconfigurations**: Sudo abuse, Cron jobs, Kernel Exploits, Insecure Permissions
